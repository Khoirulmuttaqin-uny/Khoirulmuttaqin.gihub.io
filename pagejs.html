JavaScript

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman tingkat tinggi dan dinamis. JavaScript populer di internet dan dapat bekerja di sebagian besar penjelajah web populer seperti Google Chrome, Internet Explorer, Mozilla Firefox, Netscape dan Opera. Kode JavaScript dapat disisipkan dalam halaman web menggunakan tag SCRIPT.

Tahukah kamu bahwa JavaScript adalah bahasa pemrograman yang digunakan dalam pengembangan website agar lebih dinamis dan interaktif. Kalau sebelumnya kamu hanya mengenal HTML dan CSS, nah sekarang kamu jadi tahu bahwa JavaScript dapat meningkatkan fungsionalitas pada halaman web. Bahkan dengan JavaScript ini kamu bisa membuat aplikasi, tools, atau bahkan game pada web.

Fungsi JavaScript

Fungsi Javascript kira-kira meliputi pembuatan aplikasi mobile, desktop, web, game, sampai membuat slide presentasi. Meskipun fungsi Javascript cukup banyak, biasanya bahasa pemrograman ini digunakan untuk aplikasi web. Hal ini karena website bisa terlihat lebih menarik dengan Javascript

Javascript memang cocok untuk mengembangkan website. Namun, bahasa pemrograman ini juga bisa dipakai untuk membuat aplikasi mobile. Biasanya, pembuatannya menggunakan framework.

Framework JavaScript

Sepenting apa sih framework Javascript untuk programmer web? Seperti yang sudah kita ketahui di artikel sebelumnya, framework merupakan sebuah tools yang penting dalam pengembangan serta mendesain sebuah website. Karenanya, framework dipilih oleh banyak sekali programmer web untuk mempermudah pekerjaan mereka.

Framework Javascript memungkinkan seorang programmer untuk membuat halaman webnya menjadi responsif atau mobile-friendly. Responsif menjadi alasan lain pentingnya framework Javascript untuk programmer web.
